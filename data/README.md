# Data Directory

## raw

외부 기관 또는 공개 데이터 서비스에서 내려받은 원본 데이터입니다.

- `online_sales_by_channel_2025.csv`: 판매 매체별 상품군 거래액
- `overseas_direct_sales_by_region.csv`: 지역별 상품군 해외 직접판매액
- `temperature_multi_year_monthly.csv`: 여러 연도의 월별 평균기온
- `temperature_seoul_monthly_2025.csv`: 서울의 2025년 월별 평균기온

## interim

- `merged_naver_click_sales_quarterly_2025_no_demographics.csv`:
  분기 클릭 지수와 판매액을 병합한 중간 데이터

## processed

- `cleaned_online_sales.csv`: 전처리된 온라인 판매 데이터
- `final_naver_click_sales_quarterly_2025.csv`: 최종 분기 클릭 및 판매 분석 데이터
- `google_trends_region_2021_2025.csv`: 지역별 Google Trends 결과
- `naver_shopping_3depth_monthly_2025.csv`: 상품별 월간 클릭 지수
- `naver_shopping_3depth_segment_share_wide_2025.csv`:
  상품별 연령대, 성별, 기기 이용 비율
